# 설정

이 문서는 ZAI 인증 플러그인 설정 방법을 설명합니다.

## 설정 파일

설정 파일 위치: `~/.config/opencode/zai.json`

## 옵션

| 옵션 | 타입 | 기본값 | 설명 |
|--------|------|---------|-------------|
| debug | boolean | false | 디버그 로깅 활성화 |

## 환경 변수

| 변수명 | 덮어쓰기 대상 | 설명 |
|----------|----------|-------------|
| ZAI_DEBUG | debug | 설정 파일의 `debug` 설정을 덮어씁니다. |
| ZAI_API_KEY_0~N | - | 로테이션을 위한 동적 키 감지. |

## 우선순위

설정은 다음 우선순위에 따라 로드됩니다 (가장 높은 것이 낮은 것을 덮어씀):

1. 환경 변수
2. 설정 파일 (`~/.config/opencode/zai.json`)
3. 기본값

## 설정 예시

```json
{
  "debug": false
}
```
